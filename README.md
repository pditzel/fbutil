## fbutil


### Beschreibung

Bash-Skript zum Steuern und Abfragen einer Fritzbox

### Erweiterungen

Module sind bash-Funktionen die in einzelnen Dateien im
"lib/modules"-Verzeichniss liegen. Jedes Modul hat eine Konfigurationsdatei im
Verzeichnis "fbutil.cfg.d"mit dem selben Namen wie das Modul. Erst wenn eine
Konfigurationsdatei angelegt ist wird das Modul geladen.

Die Steuerung erfolgt ausschließlich über die Konfigurationsdateien.
